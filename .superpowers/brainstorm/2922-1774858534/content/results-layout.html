B. Hero Number + Inline Charts

Epoch time dominates as the hero metric (it's what leadership cares about most). Secondary KPIs inline below. All charts visible simultaneously, no tabs. More editorial, less dashboard.

Recommendation: A — the 4 equal-weight KPI cards give each metric appropriate prominence. Tabs keep the chart area clean. Note: Impeccable says "DON'T use the hero metric layout template" — Option B risks that anti-pattern.
